# Pi Extensions Manifest
# Add one extension per line. Lines starting with # are comments.
#
# Supported formats:
#   git:github.com/user/repo@version
#   npm:@scope/package@version
#   /absolute/path/to/extension
#   ./relative/path/to/extension
#
# Examples:
#   git:github.com/xRyul/pi-nvidia-nim
#   npm:@some/package@1.0.0
